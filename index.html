Dica: No GNU/Linux, você pode converter o formato hexmem para o formato MEM do Neander usando o comando xxd, que acompanha o editor de texto Vim.
– Para converter de hexmem para mem: salve o código acima em um arquivo texto e use:
xxd -r -p arquivo.hex >arquivo.mem
– Para converter de mem para hexmem:
xxd -p arquivo.mem >arquivo.hex
